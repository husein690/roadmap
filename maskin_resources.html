⚙️ أوامر Assembly الأساسية
Data Movement:
MOV eax, 5 ; eax = 5 MOV eax, [ebx] ; eax = قيمة في العنوان ebx MOV [ebx], eax ; احفظ eax في عنوان ebx LEA eax, [ebx+4] ; eax = العنوان نفسه (مش القيمة!) XCHG eax, ebx ; تبديل قيم الـ registers
Arithmetic:
ADD eax, ebx ; eax = eax + ebx SUB eax, ebx ; eax = eax - ebx MUL ebx ; EDX:EAX = EAX * ebx (unsigned) IMUL ebx ; EDX:EAX = EAX * ebx (signed) DIV ebx ; EAX = EDX:EAX / ebx, EDX = remainder INC eax ; eax++ DEC eax ; eax-- NEG eax ; eax = -eax
Logical:
AND eax, ebx ; bitwise AND OR eax, ebx ; bitwise OR XOR eax, eax ; eax = 0 (أسرع طريقة لـ zero) NOT eax ; bitwise NOT SHL eax, 2 ; shift left 2 = × 4 SHR eax, 2 ; shift right 2 = ÷ 4
Control Flow:
CMP eax, ebx ; مقارنة (تعدّل الـ Flags) JMP label ; jump دايماً JE label ; jump if equal (ZF=1) JNE label ; jump if not equal (ZF=0) JL label ; jump if less (signed) JG label ; jump if greater (signed) JB label ; jump if below (unsigned) JA label ; jump if above (unsigned)
Stack:
PUSH eax ; ESP -= 4, [ESP] = eax POP eax ; eax = [ESP], ESP += 4 CALL func ; PUSH EIP ثم JMP func RET ; POP EIP (ارجع للمستدعي)
🔄 مثال — Loop في Assembly
; for (int i = 0; i < 10; i++) sum += i; MOV ecx, 0 ; i = 0 MOV eax, 0 ; sum = 0 loop_start: CMP ecx, 10 ; i < 10? JGE loop_end ; لو i >= 10 اخرج ADD eax, ecx ; sum += i INC ecx ; i++ JMP loop_start ; كرر loop_end: ; eax = 45 (مجموع 0..9)

// Pointers int x = 42; int *p = &x; // p يحمل عنوان x *p = 100; // تغيير قيمة x عبر الـ pointer // Arrays int arr[5] = {1,2,3,4,5}; // arr = عنوان أول عنصر (arr == &arr[0]) // Pointer Arithmetic int *ptr = arr; ptr++; // ينتقل 4 bytes (حجم int) *(ptr + 2) // نفس arr[2] // Strings char str[] = "Hello"; // [H,e,l,l,o,\0] char *s = str;
💾 Dynamic Memory
// Heap allocation: int *arr = (int*) malloc(5 * sizeof(int)); // احكي دايماً على NULL check! if (arr == NULL) { /* handle error */ } arr[0] = 10; arr[1] = 20; // Resize: arr = (int*) realloc(arr, 10 * sizeof(int)); // Free — لازم دايماً! free(arr); arr = NULL; // تجنب dangling pointer
⚠️ Memory Leaks: كل malloc يحتاج free. Dangling Pointer: استخدام pointer بعد free!

📚 Stack Frame — كيف تشتغل الدوال
; استدعاء دالة: int add(int a, int b) ; 1. المستدعي يدفع الـ arguments (من اليمين للشمال): PUSH b ; argument 2 PUSH a ; argument 1 CALL add ; PUSH EIP + JMP add ; 2. داخل الدالة: PUSH EBP ; احفظ EBP القديم MOV EBP, ESP ; ابني الـ stack frame SUB ESP, 8 ; مساحة للـ local variables ; الوصول للـ arguments: MOV eax, [EBP+8] ; a MOV ebx, [EBP+12] ; b ; 3. العودة: MOV eax, result ; النتيجة في EAX MOV ESP, EBP ; أزل الـ locals POP EBP ; استرجع EBP RET ; POP EIP + ارجع
📊 Stack Frame Layout
High addresses ┌─────────────────┐ │ arg b │ EBP+12 │ arg a │ EBP+8 │ return addr │ EBP+4 ← CALL حفظه │ old EBP │ EBP ← PUSH EBP حفظه │ local var 1 │ EBP-4 │ local var 2 │ EBP-8 ← ESP └─────────────────┘ Low addresses
💡 Calling Convention (cdecl): Arguments من اليمين للشمال، المستدعي ينظف الـ Stack، النتيجة في EAX
